Hexo常用命令

发表于 2016-01-02   |   分类于 TECH   |  

新建文章

hexo new 文章标题
或者
hexo n 文章标题

生成静态文件

hexo generate
或者
hexo g

预览

hexo server
或者
hexo s

访问网址： http://localhost:4000/

发布

hexo deploy
或者
hexo d
或者
hexo d -g # g=generate，发布之前生成静态文件

前提是，在配置文件 _config.yml 里已经配置了 deploy 节点。

删除文章

注：
hexo命令，后面的一个单词基本都可以简写成第一个字母。前提是同一个字母下，只能有一个命令。
举个反例：
hexo clean 不能简化成 hexo c，因为c开头的有两个命令hexo clean 和hexo config 。
这时候多敲一个字母就可以了，hexo cl 。

更多更详细的命令请参考Hexo文档 。
